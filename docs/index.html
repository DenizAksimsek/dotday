A human readable calendar format.

Compiles to iCalendar (.ics) files.

One file = one day.

Designed for ease of parsing, both by humans and by computers.

The GitHub repo contains an informal description and a reference implementation.

Example 

; 2020-07-13.day
; comment
; The title is the date.
= 2020-07-14 =

; All day event
--
International Nothing Day

; Events are separated by blank lines
; !1h -> Your calendar app will remind you 1 hour before the event
10:00 -- 11.50 !1h
MATH 102
@ DB18
i hate this course
so much
~ Mike Oxmall <mike.oxmall@example.edu>

14.00 -- 15.50
HIST 102
& https://public.denizaksimsek.com/metro-map/metro5.svg
@ D028

You can subscribe to the iCalendar feed for the above example.

FAQ (Fairly-likely-to-be-Asked Questions)

"Why did you make this?"
— Someone, eventually

Because nobody else had made it. I wanted a text based calendar that I could put on my desktop and phone, but I also wanted notifications on my phone (text files are too polite to bother you like that).
