A human readable calendar format.

Compiles to iCalendar (.ics) files.

One file = one day.

Designed for ease of parsing, both by humans and by computers.

The GitHub repo contains an informal description and a reference information in JavaScript that outputs iCalendar data.

Example 

; 2020-07-13.day -- the filename is the date
; comment
; The title is optional.
13 July (Monday)
